# Gomi App

## Installation
1. Clone this repository:
  ```sh
  git clone https://github.com/andrizan/gomi-app.git
  cd gomi-app
  ```
2. Build and run the application:
  ```sh
  go run main.go
  ```

## Requirements
- Go 1.18 or newer
- Fyne toolkit
